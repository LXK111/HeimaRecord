## 打开方式

进入 `heima-record` 文件夹，双击 `index.html`。该文件已经内置页面样式和程序代码，不需要 Node.js。

推荐浏览器：

- Chrome
- Edge
- Safari

## 导入场次

点击“导入”，选择 Excel 或 CSV 文件。

导入文件首行建议包含：

- 场次编号
- 组别
- 场地
- 红方姓名
- 红方单位
- 蓝方姓名
- 蓝方单位

也可以参考同目录下的 `示例导入模板.csv`。

## 比赛记录

进入“场次”，选择比赛后进入“控制台”。

控制台支持：

- 开始、暂停、重置计时
- 红方、蓝方加减分
- 处罚扣分
- 手动判定红方胜、蓝方胜或平局
- 自动记录操作日志

## 本地保存

系统会自动保存到当前浏览器的本地数据中。

刷新页面、关闭后重新打开，通常不会丢失数据。

注意：

- 清理浏览器站点数据会删除本地记录
- 更换浏览器或更换电脑不会自动带走数据
- 比赛现场建议定期导出完整备份 JSON

## 完整备份和恢复

进入“结果”，使用“完整备份”区域：

- “导出备份 JSON”：保存规则、场次、比分、计时状态和操作记录
- “导入备份 JSON”：恢复之前保存的完整状态

完整备份用于恢复现场状态；成绩 Excel/CSV 只用于提交结果。

## 结果导出

进入“结果”，可以导出：

- CSV 成绩表
- Excel 成绩表

比赛结束后必须导出结果文件作为正式留档。

## 无 Node 说明

这个交付包运行时不需要 Node.js、npm、数据库或网络。

开发和重新打包仍需要开发环境；比赛现场只需要浏览器。
