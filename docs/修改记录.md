## 规则文件设计

优先使用一个 XLSX 文件，包含 4 个工作表：

- `基础规则`：配置计分模式、比赛时长、目标分、最大回合数、是否允许双方得分、是否允许无效回合等。
- `部位分值`：配置部位 ID、部位名称、分值、是否启用。
- `警告分级`：配置警告 ID、警告名称、扣分、是否计处罚、是否直接判负、是否中止比赛，以及中止后的胜负或平局关系。
- `警告转换`：配置某类警告累计多少次后转换为另一类警告。

CSV 兼容规则：

- 单个 CSV 表按表头识别类型。
- 表头包含 `部位ID` 时导入部位分值。
- 表头包含 `警告ID` 时导入警告分级。
- 表头包含 `来源警告ID` 时导入警告转换。

## 验收清单

- 导入场次后可正常进入控制台。
- 点击不同部位后按规则分值增加比分。
- 切换为限制回合模式后，记录回合结果，时间到或回合数打满结束比赛。
- 点击警告后按警告分级扣分、计处罚、触发转换和判负。
- 点击记录申诉后日志包含剩余时间和当前比分。
- 点击撤销后恢复上一步操作前状态并保留撤销日志。
- 点击重置比赛后清空比分、警告、处罚、胜方和时间，保留日志。
- 比赛结束后普通操作锁定，赛后修正可改比分和胜方。
- 结果导出包含警告、申诉和赛后修正记录。
- `npm run lint` 和 `npm run release` 通过。
