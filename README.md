## 项目定位

`heima-record` 是一个本地 Web 端兵击赛事控制台，覆盖选手导入、赛事编排、现场计时记分、规则判定、本地保存和结果导出。

## 当前能力

- 导入 Excel / CSV 场次表
- 配置比赛时长、目标分、平局、加时和处罚扣分规则
- 单场比赛控制台：计时、加减分、处罚、手动结束
- 浏览器 IndexedDB 本地保存
- 导出 CSV / Excel 比赛结果
- 导出 / 导入完整 JSON 备份
- 通过 CSV / XLSX 配置部位分值、警告分级和警告转换
- 支持申诉记录、撤销、重置比赛和赛后修正
- 支持目标分模式和限制回合模式
- 支持综合判定、判罚中止比赛结果配置和加时
- 支持小组循环 + 单败、瑞士轮 + 单败、直接单败和双败淘汰赛
- 支持排名规则优先级、纪律扣分、附加赛和季军赛
- 导出赛事摘要、最终名次、预赛排名、淘汰签表和全部场次

## 导入字段

导入文件首行建议包含以下字段，中文或英文表头均可：

- 场次编号 / matchNo
- 组别 / group
- 场地 / piste
- 红方姓名 / redName
- 红方单位 / redClub
- 蓝方姓名 / blueName
- 蓝方单位 / blueClub

## 本地运行

```bash
npm install
npm run dev
```

赛事领域验收测试：

```bash
npm run test
```

## 生成无 Node 交付包

```bash
npm run release
```

生成目录：

```text
release/heima-record/
```

比赛现场使用者只需要浏览器，双击发布包里的单文件 `index.html` 即可打开。

发布包同时包含使用说明、赛事验收清单、场次示例和规则配置模板。`npm run release` 会在打包后自动检查这些交付文件。
